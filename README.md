# Obsidian Notes

This repository contains Yosi's public access Obsidian notes.

## About

These are personal notes created and maintained using [Obsidian](https://obsidian.md/), a powerful knowledge base that works on top of a local folder of plain text Markdown files.

## Structure

Notes are organized in a way that makes sense for personal knowledge management, covering various topics and ideas.

## Usage

Feel free to browse through the notes. They are publicly accessible for reference and sharing purposes.
